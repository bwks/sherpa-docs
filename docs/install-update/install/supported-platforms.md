# Supported Platforms

The Sherpa platform has both client and server applications.

## Sherpa Server

Sherpa server is available for Linux platforms only.

## Sherpa Client

Binaries are available for [Linux](), [Mac]() and [Windows]() compiled for x86_64 systems.

## Host Machine

Sherpa server is tested with Intel `i9` and `xeon` series x86_64 CPU's.

## Host Operating System

Sherpa is tested on the most recent version of Ubuntu server LTS.
The current LTS version under test is `24.04` and will move to `26.04` once it's released.

## Virtualization Provider

QEMU/KVM/Libvirt

## Container Provider

Docker Engine
